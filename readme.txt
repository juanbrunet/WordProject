WordProject is a Project Management system with a simple core function and a set of basic plugins that extends its funcionality.

== Description ==

WordProject integrates a Project Management based in Task, Risk and Milestones that allows you control each project in a basic way adding some information 
like calendar, resources and indicators. This plugin can be extended with a set of free plugins who works whit this data and can be managed in a plugin area.

Major new features in WordProject 1.0 include:

* A simple Project Manager with tasks, risks and milestones arranged in sections. 
* Resource Management for assign roles inside the project.
* Calendar View for items in a day.
* Notification system

== Installation ==

Upload the Akismet plugin to your blog, activate it and start projecting
